# 安装
## Windows
下载https://github.com/kde-yyds/FuckHfr/releases/download/v1.0/fuckhfr-bin.zip
解压并运行fuckhfr.exe
## GNU/Linux
```
git clone --depth=1 https://github.com/kde-yyds/FuckHfr
cd FuckHfr
mkdir build
cd build
cmake .. -DCMAKE_INSTALL_PREFIX=/usr -G Ninja
ninja -j8
sudo ninja install
```
然后运行`fuckhfr`
